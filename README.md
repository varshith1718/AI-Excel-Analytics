#  AI Excel Analytics & Report Generator

An AI-powered Python application that analyzes any Excel workbook and automatically generates data insights, charts, reports, and PowerPoint presentations.

---

##  Features

- Upload any Excel (.xlsx) file
- Supports multiple worksheets
- Automatic dataset analysis
- Missing value detection
- Duplicate record detection
- Summary statistics generation
- AI-generated insights
- Executive recommendations
- Automatic chart generation
- Professional PowerPoint presentation
- Downloadable analytics report

---

## Technologies Used

- Python 3.10+
- Pandas
- OpenPyXL
- Matplotlib
- Streamlit
- python-pptx

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Excel-Analytics.git
```

Go to the project folder

```bash
cd AI-Excel-Analytics
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

##  Run the Application

### Streamlit

```bash
streamlit run app.py
```

### Console Version

```bash
python main.py
```
